How to Use

audAIence is an innovative platform that enhances classroom interaction using AI. Here's how to get started:

  1. For students, join a class by navigating to: {{ url }}/[class_id]
  2. For teachers, access the teacher interface at: {{ url }}/[class_id]/teacher
  3. Teachers can upload slides and monitor student engagement in real-time.
  4. Students can ask questions and indicate their understanding using color-coded feedback.

Experience a new level of classroom interaction with audAIence!
